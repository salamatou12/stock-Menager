### Description du projet

Ce projet consiste à réaliser une interface complète de gestion des commandes et du stock pour des produits électroniques.
L’objectif est de proposer un tableau de bord moderne, responsive et fidèle à une interface professionnelle, permettant d'afficher les commandes, le client, les produits et les actions associées.

L’interface est entièrement créée en HTML et CSS uniquement, sans JavaScript.

## Objectifs du projet

Créer un dashboard clair et ergonomique pour la gestion du stock.

Afficher des cartes de commandes avec statut, produits, prix et actions.

Respecter une structure visuelle moderne inspirée des outils professionnels.

Utiliser uniquement HTML + CSS pour la structure et le design.

Produire un code organisé, propre et facile à modifier.

##Technologies utilisées

HTML5 – Structure du contenu

CSS3 – Styles, layout, responsive

Font Awesome – Icônes du tableau de bord

##Fonctionnalités principales

Sidebar de navigation fonctionnelle

Header avec icône notifications + profil administrateur

Liste de commandes affichée sous forme de cartes

Statuts : En cours, Expédiée, Livrée

Boutons d’actions : Voir détails, Modifier, Suivre, Expédier, etc.

Affichage du prix + quantité + produit + client

Design moderne, minimaliste et facile à comprendre
